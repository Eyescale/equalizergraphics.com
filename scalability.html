Survey

Applications build with Equalizer are inherently scalable, aggregating multiple graphics cards, processors and computers to render a single or multiple views. The application's rendering code is parallelized, in contrast to most other solutions which distribute the OpenGL command stream produced by a single application thread.

Equalizer provides a comprehensive set of algorithms to parallelize, balance and optimize the rendering of demanding data sets:
