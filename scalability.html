Scalability

Equalizer implements a wide range of decomposition schemes to distribute the rendering for a single view across multiple resources. The following decomposition schemes are possible:

  • 2D decomposes the rendering in screen-space, that is, each contributing rendering unit processes a tile of the final view. Equalizer automatically recomposes the tiles side-by-side on the destination view.
  • DB decomposes the rendered database so that all rendering units process the scene in parallel.
