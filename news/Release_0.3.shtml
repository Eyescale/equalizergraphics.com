Using two GPU's in parallel on Windows XP

We are pleased to announce the release of Equalizer 0.3, a framework for the development and deployment of scalable OpenGL applications. Equalizer 0.3 now also runs on Windows XP, extending the support to all major operating systems. Various new features and stability improvements make this release the fastest and most mature Equalizer version so far. The stability and performance of this release has been verified on a 16-node, 32-GPU cluster.

The full release notes for version 0.3 are available at
www.equalizergraphics.com/documents/RelNotes/RelNotes_0.3.0.html.

Commercial support, custom software development and porting services are available from Tungsten Graphics. Please contact sales@tungstengraphics.com for further information.
