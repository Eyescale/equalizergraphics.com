Using two GPU's in parallel on Windows XP

We are pleased to announce the release of Equalizer 0.3, a framework for the development and deployment of scalable OpenGL applications. Equalizer 0.3 adds support for Windows XP, among with various minor features and stability improvements. The stability and performance of this release has been tested for the first time on a 16-node, 32-GPU cluster.

More information about Equalizer can be found at www.equalizergraphics.com. The release notes for version 0.3 are available at www.equalizergraphics.com/documents/RelNotes/RelNotes_0.3.0.html.

Commercial support, custom software development and porting services are available from Tungsten Graphics. Please contact sales@tungstengraphics.com for further information.
