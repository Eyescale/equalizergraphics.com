Equalizer Architecture

Equalizer Server

The Equalizer server is managing the resources of a visualization system. It is configured to be aware of the system's compute and graphics resources, projection systems and hardware topology. Based on this information, it dynamically allocates the resources for the clients and deploys these resources optimally based on the current load of the system.

Equalizer Client Library

The Equalizer client library is the applications interface to the server. An Equalizer application declares the desired views. Available views on the visualization system can be queried from the server. Fullscreen views are reserved for exclusive usage. declare the desired views, describe their rendering characteristics and provide the rendering client.

Equalizer Tools

Configuration tools aid during the development of the Equalizer server configuration. They automatically discover parts of the visualization system's resources and their topology. Server maintenance tools modify the server's state at runtime, e.g., to setup a new view configuration. Analysis and statistics tools
