Developers actively developing Equalizer-based applications should either use the subversion source code tree, a nightly build or the latest developer release.

End users or programs not in active development should use the latest major release.

The project is hosted on sourceforge.net. If you are using Equalizer, please fill out our survey and receive a free download of the latest Equalizer programming guide, also available as a book!
