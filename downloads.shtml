Equalizer

No stable releases of Equalizer are available so far. Interested developers can access the project information or download one of the prerelease versions below. If you plan contributions to Equalizer, please get in touch with us.

Subversion Repository

Follow the instructions to check out the source code on the sourceforge project page, or just check out the code from https://equalizer.svn.sourceforge.net/svnroot/equalizer/trunk using subversion. Please check out trunk only, not the top-level repository as explained on sourceforge, unless you want access to all previous releases. A local, read-only copy is available for browsing the source code. The repository has the following layout and size: 

 24M trunk/src
165M trunk/doc
206M trunk/website

396M tags/beta-0.3
395M tags/alpha-0.3

GLAsync

GLAsync is a small, independent C++ class delivering asynchronous OpenGL downloads in the spirit of the SGIX_async extension. The ARB_sync extension provides now a better way for asynchronous operations.
