Equalizer

No stable releases of Equalizer are available so far. Interested developers can access the project information or download one of the prerelease versions below. If you plan contributions to Equalizer, please get in touch with us.

Subversion Repository

Follow the instructions to check out the source code on the sourceforge project page, or just check out the code from https://equalizer.svn.sourceforge.net/svnroot/equalizer/trunk using subversion. Please check out trunk only, not the top-level repository as explained on sourceforge, unless you want access to all previous releases. The repository has the following layout and size: 

137M trunk/doc
 14M trunk/src
188M trunk/website
294M tags/release-0.1
339M tags/release-0.2

Version 0.2

Equalizer 0.2 adds support for scalable rendering, by using multiple CPU's, GPU's and computers to scale the rendering performance of a single view. Release Notes can be found in the Documentation section. The source code is available for download locally and from Sourceforge.

Version 0.1

Equalizer 0.1 is the first official release. It is intended as a preview and evaluation snapshot for application developers and early adopters. Release Notes can be found in the Documentation section. The source code is available for download locally and from Sourceforge.

GLAsync

GLAsync is a small, independent C++ class delivering asynchronous OpenGL downloads in the spirit of the SGIX_async extension. The ARB_sync extension provides now a better way for asynchronous operations.

Links: Documentation, Changelog, Latest Version.
