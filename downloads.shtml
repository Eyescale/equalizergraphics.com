Equalizer

No stable releases of Equalizer are available so far. Interested developers can access the project information or download one of the versions below. If you plan contributions to Equalizer, please get in touch with us.

Subversion Repository

Follow the instructions to check out thesource code on the sourceforge project page, or just check out the code from https://svn.sourceforge.net/svnroot/equalizer/trunk using subversion. Please check out trunk only, not the top-level repository as explained on sourceforge, unless you want access to all previous releases.

Version 0.1

Equalizer 0.1 is the first official release. It is intended as a preview and evaluation snapshot for application developers and early adopters. Release Notes can be found in the Documentation section. The source code is available for download from sourceforge.

GLAsync

GLAsync is a small, independent C++ class delivering asynchronous OpenGL downloads in the spirit of the SGIX_async extension. The ARB_sync extension provides now a better way for asynchronous operations.

Links: Documentation, Changelog, Latest Version.
