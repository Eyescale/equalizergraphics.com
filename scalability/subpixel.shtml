Subpixel compounds decompose the rendering into different jittered frusta so that the recomposition anti-aliases the final rendering. The Subpixel compound is also compatible with applications doing their own software anti-aliasing.

As for the DB compound, the subpixel recomposition has linear increasing IO requirements for the pixel transfer with the difference that only color information has to be transmitted.
