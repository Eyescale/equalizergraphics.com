Git Repository

The git repository gives access to the latest source code, and should be used by developers modifying the Equalizer source code or interested in the latest features. The easiest way is to build using the CMake superproject, as described in the Readme. We try to keep the git master stable and encourage developers to use git, or at least a developer version.

We encourage you to fork the repository on github, code away and push the changes to your fork. If we don't pick them up automatically within a week, please send us a pull request.
