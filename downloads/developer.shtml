Developer releases are regular, minor versions in-between major releases. Typically we tag a developer release after a feature has been implemented and tested. This ensures a reasonable stability, while providing early access to new features.

Developer releases can be checked out from the subversion repository or by downloading the source code below. The following developer releases, based on the last major release, are available:
