Developer releases are regular, minor versions in-between major releases. Typically we tag a developer release after a feature has been implemented and tested. This ensures a reasonable stability, while providing early access to new features. The minor version of developer releases is odd (e.g. 1.1.0) while major releases have even minor revisions (e.g. 1.2.1).

Developer releases can be checked out from github or by downloading the source code below. The following developer releases, based on the last major release, are available:
