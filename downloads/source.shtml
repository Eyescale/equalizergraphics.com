Subversion

The git repository gives access to the latest source code, and should be used by developers modifying the Equalizer source code or interested in the latest features. The tree can be checked out from github. We try to keep the svn tree stable and encourage developers to use the svn tree, or at least a developer version.

The repository has the following layout and clone size: 

49M Equalizer
272M EqDocs
939M equalizergraphics.com
