Git Repository

The git repository gives access to the latest source code, and should be used by developers modifying the Equalizer source code or interested in the latest features. The tree can be checked out from github. We try to keep the git master stable and encourage developers to use git, or at least a developer version.

Versions up to and including 1.0 were developed using a Subversion repository.

The repository has the following layout and clone size: 

49M Equalizer
272M EqDocs
939M equalizergraphics.com
