Subversion

The subversion repository gives access to the latest source code, and should be used by developers modifying the Equalizer source code or interested in the latest features. The tree can be checked out using subversion. We try to keep the svn tree stable and encourage developers to use the svn tree, or at least a developer version.

Make sure you check out trunk/src only, not the top-level repository as explained on sourceforge. Otherwise you will get all previous releases, well over one gigabyte of data.

A local, read-only copy is available for browsing the source code. The repository has the following layout and size:

202M trunk/doc
